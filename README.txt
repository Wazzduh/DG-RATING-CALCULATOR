# Disc Golf Rating Calc PWA

Files included:
- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `icons/`

## How to install on your Google Pixel 10 Pro
1. Upload these files to any HTTPS web host.
2. Open the hosted `index.html` in Chrome on your Pixel.
3. Tap the browser menu.
4. Tap **Install app** or **Add to Home screen**.

## Easy hosting options
- GitHub Pages
- Netlify
- Vercel
- Any web host that serves HTTPS

## Notes
This package is an installable PWA, not a native APK.
